Taaki layering par poora control mil sake. ================================================= 📝 z-index KYA SIRF SIBLINGS PAR WORK KARTA HAI? ================================================= Short Answer: ❌ Nahi. Ye aadha sach hai. Sahi statement: z-index same stacking context ke elements ke beech work karta hai. ================================================= BEGINNER RULE ================================================= Abhi ke liye yaad rakho: z-index zyadatar siblings ke beech kaam karta hai. Ye 90% cases me sahi niklega. ================================================= ADVANCED RULE ================================================= Technically: Sibling hona zaruri nahi. Same stacking context me hona zaruri hai. ================================================= Example 1: Siblings =================================================
CSS: .box1{ position:absolute; z-index:1; } .box2{ position:absolute; z-index:2; } Result: box2 ↑ box1 Kyun? Kyuki dono: ✅ Positioned hain ✅ Same stacking context me hain ✅ Siblings hain ================================================= Example 2: Nested Elements =================================================
CSS: .box1{ z-index:1; } .child{ z-index:9999; } .box2{ z-index:2; } Question: Kya child box2 ke upar aa jayega? Answer: ❌ Zaruri nahi. ================================================= Kyun? ================================================= Structure: parent ├── box1 (z-index:1) │ └── child (z-index:9999) │ └── box2 (z-index:2) Browser pehle compare karega: box1 vs box2 Yaha: box2 > box1 Isliye: child ka z-index 9999 hone ke baad bhi wo box2 ke neeche reh sakta hai. ================================================= IMPORTANT LESSON ================================================= Child apne parent ke stacking context ke andar phansa rehta hai. Wo bahar jaakar dusre stacking context se directly fight nahi karta. ================================================= 🔥 GOLDEN RULE ================================================= z-index directly siblings par nahi, balki same stacking context wale elements par work karta hai. ================================================= EXAM / INTERVIEW ANSWER ================================================= "z-index works on positioned elements within the same stacking context." Ye line yaad rakh lo. Ye professional answer hai.

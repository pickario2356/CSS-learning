# 🌌 Stellar

A modern, responsive space-themed landing page built using HTML and CSS. This project focuses on clean layouts, responsive design, smooth UI interactions, and polished visual aesthetics without relying on JavaScript.

## ✨ Features

- Responsive navigation bar
- Modern hero section
- Staggered feature/gallery layout
- CSS Grid & Flexbox layouts
- Custom typography
- Hover animations and transitions
- Fully responsive design
- Clean and organized code structure

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
- Material Symbols
